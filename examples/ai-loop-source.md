# AI Loop：智能体持续反馈闭环

AI Loop 不是传统程序中的简单循环，而是 AI 执行、获得结果、评估、调整、再次执行并持续优化的反馈闭环。

## Agent Loop

智能体从用户目标出发，理解任务并制定计划，然后调用搜索、API、代码或业务系统等工具。执行后读取结果和错误，判断是否完成；如果信息不足或质量不合格，则调整策略并进入下一轮 Reason、Act、Observe、Evaluate。

## 自我改进与 Coding Loop

AI 生成代码后运行测试，根据错误日志修正代码并重新测试，直到通过。生成、评价、修正和重新生成构成自我改进循环。

## 企业自动化 Loop

客户需求或业务事件触发 AI 决策，智能体创建任务、调用系统接口、检查结果、通知人员并持续跟踪状态。事件、决策、动作和反馈形成自动化闭环。

## Memory Loop

智能体从对话和任务中提取重要信息，保存用户偏好、历史和经验，在未来任务中调用记忆并调整行为。

## Human-in-the-loop

低风险动作可以自动执行，高金额、敏感权限或不确定操作必须等待人工确认。人工反馈进入下一轮决策，保证系统安全可控。

一个稳定的 Agent Loop 应覆盖目标管理、任务拆解、工具调用、状态观察、结果评估、记忆更新、自我修正、人工确认与最终交付。
